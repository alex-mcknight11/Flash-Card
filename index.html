JavaScript

JavaScript is a programming language used to make web pages interactive.

Operators

JavaScript operators are used to assign values, compare values, perform arithmetic operations and more

Variables

Variables are containers for storing data, they are always declared using var, let or const

Variable naming conventions

Variables are named using camelCase, and all variable names start with a letter. Ex: firstName = "George";

Functions

Functions are a block of code designed to perform a particular task. A JavaScript function is executed when "something" invokes or calls it.

Methods

Methods are built in functions that can be performed on data objects/variables. Ex: string = "hello"; string.toUpperCase() = "HELLO"

Arguments

Arguments are the named values that are passed into a function in order for them to be manipulated by the function

Parameters

the names of values passed to (and received by) the function.

Return

A statement that stops the execution of a function and returns a value.

Chaining methods

in Jquery, you can append the action to the previous action within a single statement.

Strings

Zero or more characters written inside quotation marks.

Booleans

A code type that can only take two values: True or False.

Undefined

a variable with no value.

NaN

A non legal number, short for "Not-a-Number."

Null

An empty value.

Alert

A pop-up window running with JavaScript.

Comments

Notes about or detailing your code put in as non-functional text.

jQuery

A JavaScript library used to incorporate easy JavaScript on webpages.

Attributes

A node's specified features, e.g, a NamedNodeMap Object.
